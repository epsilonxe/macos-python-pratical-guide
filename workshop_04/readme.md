# Workshop 04

จงเขียนโปรแกรมในการแปลผลคะแนนเป็นผลการเรียน เมื่อกำหนดให้

| คะแนน | เกรด |
| --- | --- |
| 80 - 100 | A | 
| 75 - 80 | B+ | 
| 70 - 75 | B | 
| 65 - 70 | C+ | 
| 60 - 65 | C | 
| 55 - 60 | D+ | 
| 50 - 55 | D | 
| 0 - 50 | F | 

ทั้งนี้ให้แปลผลเป็นเกรดที่สูงกว่าเสมอถ้าคะแนนดังกล่าวอยู่ในขอบบน เช่น คะแนน 80 ให้แปลผลเป็นเกรด A
